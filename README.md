# ScriptureLive AI

[![Website](https://img.shields.io/badge/Website-scripturelive.replit.app-2b7cff?logo=googlechrome&logoColor=white)](https://scripturelive.replit.app/site/)
[![Latest release](https://img.shields.io/github/v/release/nanvilow/scripturelive-ai)](https://github.com/nanvilow/scripturelive-ai/releases/latest)

Operator-grade desktop console that auto-displays Bible verses as your
preacher speaks — built for churches that broadcast over NDI to vMix,
OBS, Wirecast, or ProPresenter.

> **Visit the website:** <https://scripturelive.replit.app/site/>
> Pricing, contact, system requirements, and the Windows / macOS
> installer downloads all live there.

## What's in this repo

This is a pnpm monorepo. The pieces an operator usually cares about:

- **`artifacts/imported-app`** — the Electron + Next.js desktop app
  shipped to operators (the "ScriptureLive AI" you install).
- **`artifacts/site`** — the public marketing site published at
  [scripturelive.replit.app/site/](https://scripturelive.replit.app/site/).
- **`artifacts/api-server`** — the Replit-hosted licensing /
  telemetry / transcribe-proxy backend.

## Quick start

1. Download the installer from the
   [website](https://scripturelive.replit.app/site/) or the
   [GitHub releases page](https://github.com/nanvilow/scripturelive-ai/releases/latest).
2. Install and launch — the first-run welcome dialog walks you
   through trial activation.
3. Pick your translation, output screen, and engine preferences from
   **Settings → Help & Updates**, where you'll also find a
   **Visit website** link for sharing pricing / contact info with
   your pastor or IT lead.

For full setup, troubleshooting, and contact details, see the
[website](https://scripturelive.replit.app/site/).
